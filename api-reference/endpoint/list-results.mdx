## Overview

Retrieves a list of assessment results and grades with optional filtering by student, assignment, or class. This endpoint provides access to gradebook data and TimeBack analytics including XP points and mastery levels.

## Query Parameters

- **studentId** (optional): Filter results for a specific student
- **lineItemId** (optional): Filter results for a specific assignment/line item
- **classId** (optional): Filter results for a specific class
- **page** (optional): Page number for pagination (default: 1)
- **pageSize** (optional): Number of results per page (default: 50, max: 100)

## Usage Examples

### List All Accessible Results
Retrieve all results the user has access to:

```bash
curl -X GET "https://core.timebackapi.com/results" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Filter by Student
Get all results for a specific student:

```bash
curl -X GET "https://core.timebackapi.com/results?studentId=student-123" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Filter by Assignment
Get all submissions for a specific assignment:

```bash
curl -X GET "https://core.timebackapi.com/results?lineItemId=assignment-456" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Filter by Class with Pagination
Get results for a class with pagination:

```bash
curl -X GET "https://core.timebackapi.com/results?classId=class-789&page=1&pageSize=25" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Multiple Filters
Combine filters for specific data:

```bash
curl -X GET "https://core.timebackapi.com/results?studentId=student-123&classId=class-789" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Result Fields

### Core Fields
- **id**: Unique result identifier
- **lineItemId**: Assignment/assessment identifier
- **studentId**: Student identifier who submitted the result
- **score**: Grade or score received (string format for flexibility)
- **scoreStatus**: Current grading status
- **comment**: Teacher feedback or comments

### TimeBack Metadata
- **timeback.xp**: Experience points earned
- **timeback.masteryLevel**: Student's mastery level for this content
- **timeback.attemptCount**: Number of submission attempts
- **timeback.timeSpent**: Time spent on assignment (in seconds)

## Score Status Values

- **exempt**: Student is exempt from this assignment
- **tobeGraded**: Submission received, awaiting grading
- **partiallyGraded**: Partial feedback provided
- **fullyGraded**: Complete grade and feedback provided

## Mastery Levels

- **not-started**: Student hasn't begun the assignment
- **in-progress**: Currently working on the material
- **developing**: Basic understanding demonstrated
- **proficient**: Solid grasp of concepts
- **advanced**: Exceeds expectations
- **mastery**: Complete understanding achieved

## Access Control

- Users can only access results for students and classes within their permitted organizations
- Teachers can see results for their assigned classes
- Students can only see their own results
- Administrators have broader access based on organizational scope

## Use Cases

- **Gradebook Display**: Populate teacher gradebook interfaces
- **Student Progress**: Track individual student performance over time
- **Analytics**: Generate learning analytics and progress reports
- **Parent Reports**: Compile student progress for parent communications
- **Intervention**: Identify students needing additional support based on mastery levels
